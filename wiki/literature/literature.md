# Literature

本資料夾採 flat literature model：所有文獻頁放在同一層，使用 `topics` 與 `keywords` 控制 Obsidian 關聯圖。

Paper page 只保存單篇文獻事實、可引用 claims、閱讀限制，以及針對該單篇文章的個人評論。跨文獻比較、矛盾證據與研究判斷應寫到 [[../synthesis/synthesis|Synthesis]]。

## Metadata Rules

- `topics`: 少量高層研究領域，優先從 [[paper_topics|Paper Topics Registry]] 選取。
- `keywords`: 具體概念、方法、機制、資料集或現象。
- `reading_status`: metadata-only / abstract-only / skimmed / full-read / reproduced.
- `review_stage`: ai-extracted / human-checked / discussed / integrated / cited.
