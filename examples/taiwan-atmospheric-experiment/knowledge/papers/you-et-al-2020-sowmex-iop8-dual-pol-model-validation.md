## Reading Notes

- Research question: How well can a convection-permitting model be evaluated
  using dual-polarimetric radar parameters for a SoWMEX IOP8 squall-line case?
- Data used: NCAR S-POL dual-polarimetric radar, Taiwan radar network, WRF
  model output, radar observation operators, and ensemble data assimilation.
- Core finding: Radial wind and reflectivity assimilation help recover severe
  precipitation structure, while polarimetric variables expose microphysical
  differences that ordinary reflectivity-only verification can miss.
- Method value for RKF: A future Taiwan experiment should design model
  verification products at the same time as the observing network.

## Claims To Promote

- Claim: SoWMEX/TiMREX-era radar observations are valuable not only for event
  diagnosis but also for testing microphysics schemes and radar observation
  operators in Taiwan's subtropical environment.
  - PDF locator: title/abstract and case overview.
  - Caveat: This is a model-validation study; pair with observational
    microphysics papers before making field-operation decisions.
