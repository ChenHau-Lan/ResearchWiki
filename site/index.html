BEGINNER PATH

從 GitHub clone 到第一個研究連結

先建立自己的本機資料層，再把研究專案連到中央 RKF。專案內的 bridge 只是入口，不會複製出第二套 wiki。

閱讀快速安裝教學 →
  1. 1
    Clone & bootstrap

    取得公開 repository，建立 portable workspace 與本機 storage handles。

  2. 2
    Connect a project

    先 preview，再建立 v2 availability marker 與輕量 RKF bridge。

  3. 3
    Activate per task

    每個 Codex task 都從 OFF 開始；明確啟動後才可使用 guarded actions。

  4. 4
    Capture, read, review

    Candidate 先進 intake；只有通過成熟度與證據 gates 的內容才可升級。
