# RKF Project Skills

RKF uses a small ARS-style skill set. Modes live inside skills; the bridge to
Academic Research Skills is a protocol, not an active skill.

| Skill | Purpose |
|---|---|
| `rkf-evidence-vault` | Source capture, candidate discovery, full-text status, user-PDF handling, artifact checks |
| `rkf-knowledge-synthesis` | Paper reading drafts, knowledge objects, synthesis maturity, and topic review |
| `rkf-wiki-core` | LLM Wiki query/save/graph, paper queue/nudge, external sandbox memory |
| `rkf-lint` | Structure, reading maturity, claim boundary, graph, ARS handoff, public-safety lint |
| `rkf-connect` | Experimental shared RAW/wiki database links and external sandbox access |
