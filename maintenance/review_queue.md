## Queue Rules

- Use this file for fan-out proposals before updating multiple formal wiki
  pages.
- Use this file when a synthesis claim lacks counter-evidence or has inflated
  confidence.
- Use this file when a new source may supersede an older interpretation.
- Do not treat queue items as stable evidence.
